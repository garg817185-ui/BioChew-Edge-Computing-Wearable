Help & Documentation

Connecting via USB (Web Serial)
Click Connect USB (Serial) in the header. Your browser will show a port picker — select your Pico W's COM port. Requires Chrome or Edge with Web Serial API support.
Connecting via Bluetooth (BLE)
Click Connect Bluetooth. Make sure BLE is enabled on your device. The Pico W must be running the BioChew BLE firmware and be in pairing range.
Calibrating Detection
Adjust the Chew Signal Threshold slider based on your jaw movement strength. Lower values detect lighter chews; higher values filter out noise. The Debounce setting prevents double-counting a single chew cycle.
Using Simulation Mode
Toggle Demo Mode to test the full dashboard without hardware. Use Simulate Single Chew for manual testing or Auto-Chewing (70 CPM) for continuous simulation.
Exporting Session Data
After a session, click Export Session Data (CSV) to download a timestamped CSV with accelerometer values, gyroscope magnitude, chew count, CPM, and duration for every recorded moment.
Chewing Rate Guidelines
Research suggests 40–90 CPM is an ideal chewing pace for thorough digestion. Under 40 CPM may indicate under-chewing; over 90 CPM may suggest aggressive chewing that strains the jaw.
